## Why jolt?

Unlike Activity Monitor or other GUI tools, jolt gives you:

- **Terminal Native** — Works over SSH, in tmux, wherever you have a terminal
- **Apple Silicon Optimized** — Direct access to IOReport for accurate power metrics  
- **Lightweight** — Written in Rust, minimal resource usage
- **Scriptable** — JSON output mode for automation and integration

## Built for Power Users

jolt is designed for developers, sysadmins, and anyone who lives in the terminal. Get instant visibility into what's draining your MacBook's battery without leaving your workflow.

```bash
# Launch jolt
jolt

# Or with custom refresh rate
jolt --refresh-ms 500
```
